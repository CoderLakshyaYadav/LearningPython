## How to Run
Make sure Python 3 is installed, then run:

```zsh
python3 BasicCalculator.py
```

## Git/GitHub Practice Note
I am following a simple workflow:
1. `git status -sb`
2. `git pull --rebase origin main`
3. `git add .`
4. `git commit -m "message"`
5. `git push`
